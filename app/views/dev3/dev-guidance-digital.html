Before you upload your digital fit note

Make sure your fit note is:

  1. clear and easy to read
  2. landscape (horizontal) not portrait (vertical)
  3. all on one page with your contact details visible

If your fit note has a data matrix (a black and white square that looks like a QR code) at the bottom, it should be clear.

It's important that the photo of your fit note is clear and easy to read by our system. If it cannot be read, it could be rejected.

File size

The file size must not be larger than 15MB. If it's larger than 15MB, you will need to follow the instructions on your device to make a smaller version.

To check the file size (these steps may vary for different types of devices):

  1. Find the file on your device and select it.
  2. Select the option for more information or more details.
  3. Look for the file size - it should not be larger than 15MB or 15000kB.

File format

Your fit note must be in one of these formats:

  1. JPG
  2. JPEG
  3. HEIC
  4. HEIF
  5. BMP
  6. PNG
  7. TIFF
  8. PDF

To check the format, look at the last three letters of the file name.

For example:

  1. .pdf is a PDF file
  2. .jpg is a JPG file
